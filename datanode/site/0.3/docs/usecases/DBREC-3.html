DBREC-3.js

Excerpt

On the other hand, we identified lots of redundancy and inconsistencies in our DBpedia subset7. Especially, many links between resources are defined redundantly as http://dbpedia.org/ontology/xxx and at the same time as http://dbpedia.org/property/xxx. We then removed duplicates, leading to 1,675,711 triples, i.e. only 55.7% of the original dataset.

Synopsys

  1. having data with redundancies and inconsistencies >> access, Data, Redundancy, Inconsistency
  2. remove redundancies >> remove, Redundancy
  3. remove inconsistencies >> remove, Inconsistency
  4. remove duplicates (same as redundancies?) >> remove, Duplicate
  5. data is now 55.7 percent of before >> Data, Now, Before
