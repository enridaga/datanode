DBREC-10.js

Excerpt

Furthermore, we identified 11 useless property-value combinations to compute our recommendations, by being too generic such as rdf:type foaf:Agent. Finally, from the 767 properties used to link any resource to an artist, 336 were removed as used only to link to a single artist, and 115 were wrongly written.

Synopsys

  1. having a dataset with artists and links to resources using 767 properties >> access, Dataset, Artist, Link, Resource, 767, Property
  2. identify useless property-value combinations because too generic >> identify, Useless, PropertyValue, Combination, Because, TooGeneric
  3. identify useless properties because used to link only to a single artist. They are 336, 115 of them are wrongly written. >> identify, Useless, Property, Because, Usage, Link, Single, Artist
